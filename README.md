## Deskripsi
_______

Promptopedia is an open-source AI prompting tool for modern world to
discover, create and share creative prompts

## Features
_______

- Google Auth
- Create, Edit, and Delete Prompts
- Share Prompts
- Search for Prompts
- Prompt Tags
